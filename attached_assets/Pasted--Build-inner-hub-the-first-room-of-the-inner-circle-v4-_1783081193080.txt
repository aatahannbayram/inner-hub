# Build: inner.hub — the first room of the inner circle (v4, full specification)

## Positioning (read this first — it drives every decision)
inner.hub is **not a small community site**. It is the public face of the first room of an ambitious, deliberately-phased ecosystem ("the inner circle"): a private, invitation-only network of **AI-native app builders, founders, and investors**, started in İstanbul and aimed at the world. Future rooms (a studio, capital, and unnamed others) exist as visible-but-dim rings — teased, never explained. The site's tone is **quiet ambition**: selective like a private registry, confident like a fund, credible to developers. It must never sound like a meetup, a co-working space, or a marketing page.

Reference feeling: the archival editorial restraint of the best minimal fund sites, the developer-native credibility of radically plain "registry" pages, and the slow, expensive calm of quiet-luxury brands. Every choice must pass one test: **does it make the page feel quieter, sharper, and more inevitable?**

---

## Tech stack (use exactly this)
- **Next.js (App Router) + TypeScript + Tailwind CSS**
- **All copy server-rendered** — every word must exist in view-source HTML. No client-only text rendering.
- One long scrolling page; a hidden `/requests` admin route; a POST API route for the form.
- **Database:** Replit Postgres (or SQLite if simpler) for invitation requests.
- Allowed extra dependency: `lenis` (smooth scroll). No GSAP, no three.js, no UI kits.

## Brand system — "Quiet Signal" (locked)

### Colors (Tailwind tokens / CSS variables)
- **Ink** `#0A0A0A` — text on light; dark surfaces
- **Bone** `#F4F1EC` — paper background
- **Inner Green** `#18FF85` — the single sacred accent: logo square, `NOW` status, live elements, text selection. Never UI feedback, never decoration.
- **Stone** `#6E6A63` — secondary text, archival annotations, dim states
- Error only: muted terracotta `#B4553B` (form validation text exclusively)

Text selection: Inner Green background, Ink text.

### Typography
- **Fraunces — load as a TRUE VARIABLE FONT** via `next/font/google` with `axes: ['opsz', 'SOFT', 'WONK']`. Set `font-optical-sizing: auto` globally. Display headlines use `font-style: italic` + `font-variation-settings: 'opsz' 144, 'WONK' 1, 'SOFT' 0` — at high optical size with WONK on, the italic shows flagged ball terminals (b, d, h, k, l) that match the logo's letterforms exactly. All major statements are Fraunces italic. Never force opsz 144 below ~40px.
- **Geist** (npm `geist`; fallback Inter Tight from Google) — body, max 65ch, line-height 1.7.
- **JetBrains Mono** — labels, numbers, metadata: always uppercase, 10–13px, letter-spacing 0.08–0.12em, `font-variant-numeric: tabular-nums`.
- Type scale: hero `clamp(2.6rem, 7.5vw, 6rem)`; section headlines `clamp(1.75rem, 4vw, 3rem)`; display letter-spacing −0.01 to −0.02em; `text-wrap: balance` on all headlines.

### Logo & the green square (exact rules)
Canonical file **`inner-logo.png`** (italic serif "inner" + solid Inner Green **square** on the baseline as the period, on Ink) lives in `public/`. Use the PNG directly for favicon, OG image, and inside the Ink footer. It must never sit on Bone. On Bone, reproduce typographically: `inner` in Fraunces italic + a baseline-aligned solid green square (~0.5em) + `hub` → reads `inner■hub`. Square, never circle. In body text a normal period is fine. Optional: one slow opacity pulse on the logo square on hover only.

## Texture, motion & feel

- **Paper grain:** fixed full-viewport SVG `feTurbulence` noise (fractalNoise, baseFrequency ≈ 0.65) as a data-URI overlay at 2.5–4% opacity, `mix-blend-mode: multiply` on Bone / `screen` on Ink, `pointer-events:none`, **static** (never animated). Bone must read as paper, Ink as matte card stock.
- **One easing everywhere:** `cubic-bezier(0.16, 1, 0.3, 1)`, durations 600–900ms. Nothing bounces. No parallax, no 3D, no cursor-chasing.
- **Lenis smooth scroll**, lerp ≈ 0.1; disabled under `prefers-reduced-motion` and on touch if it fights native feel.
- **Reveals:** display headlines split by line, each line rises from an overflow mask (translateY 100%→0, 80–120ms stagger), triggered once at ~25% visibility via IntersectionObserver. Body/labels: fade + 16px rise. Section hairlines draw themselves (scaleX 0→1, origin left).
- **Preloader**, first visit per session only (sessionStorage), ≤900ms total: Ink screen, centered green square pulses once (~500ms), then a single upward wipe reveals the hero, whose line-reveal then plays. Skipped entirely under reduced motion.
- Every animated element has a reduced-motion fallback: fully visible, no transforms.

  `İSTANBUL 41.00°N 28.97°E · 14:32:07 TRT`

- Each section's mono label row is two-ended on the top hairline: label left, **archival annotation right** (both mono ~11px, Stone).

### 05 — THE ECOSYSTEM · annotation `ROOMS OPEN ONE BY ONE` — **the concentric diagram**

  - Ring 3: dimmer stroke — **capital**.
  - Ring 4 (outermost): faint **dashed** stroke — unnamed.

### 06 — REQUEST AN INVITATION · annotation `WE READ EVERYTHING`
- H (Fraunces italic): 
  > Request an invitation.
- Sub:
  > The circle is nomination-first, but good people arrive from unexpected directions. If you believe you belong here, tell us who you are.

  - `EMAIL` — email, required

- Submit (mono): `SEND` — Ink fill, Bone text; on hover the label shifts 4px right as a small **Bone** square slides in from the button's left edge (green stays sacred); on submit label becomes `SENDING…`.
- Success state replaces the form (Fraunces italic):

- Errors: inline under field, small mono, terracotta `#B4553B`. No alert boxes.

## Interaction detail pass
- Links: 1px underline draws left→right on hover (300ms, house easing).
- Logo square: single slow pulse on hover, then rest.
- Focus-visible: 2px Ink outline, 3px offset, everywhere. Keyboard: skip-to-content link, logical tab order; index rail and form fully navigable.
- No sounds, no cursors, no easter eggs except: in the served HTML, include one quiet comment near the top: `<!-- signal > noise -->`.

## Backend
- `POST /api/request`: server-side validation (required, email format, URL format), store with timestamp. Honeypot-filled submissions return generic success and are dropped. Per-IP rate limit ~5/hour if straightforward.
- `/requests`: newest-first list (date, name, email, text, link, introducer) behind a single passcode from env `ADMIN_PASSCODE`; `noindex`; no full auth system. All secrets in Replit Secrets.

## SEO & metadata
- `<title>`: `inner.hub — the inner circle`
- Description: `A private, invitation-only network of AI-native builders, founders, and investors. Started in İstanbul. Aimed everywhere.`
- OG/Twitter tags with `inner-logo.png`; favicon from the same; `lang="en"`; one `<h1>` (hero); all content crawlable in view-source.

## Performance & accessibility bar
- Lighthouse mobile: Performance ≥ 90, Accessibility ≥ 95. CLS ≈ 0 (`next/font`, reserved space, masked reveals).
- Grain, Lenis, preloader stay lean; reduced-motion disables Lenis, preloader, rotation, and all transforms.
- Contrast: Stone-on-Bone and dim-on-Ink text must still pass WCAG AA at their sizes; bump values if not.

## Definition of done
- Positioning reads as **ecosystem, not meetup**: hero + 01 + 05 carry the ambition; nothing says "small".
- Fraunces renders as a variable font; hero visibly shows opsz-144 + WONK letterform character matching the logo.
- Section 04 is a full-bleed Ink room with the 30/2/1 triptych and smooth background crossfades.
- Section 05 shows the concentric-rings diagram with slow rotation, correct labels, and the unnamed dashed outer ring; footer sub-brand line ends with `— · —`.
- Grain present (visible up close, invisible at a glance); preloader ≤900ms once per session; live İstanbul clock ticks; index rail tracks sections; archival annotations on every section label.
- Form validates, stores, confirms in Fraunces, silently drops honeypot spam; `/requests` works behind the passcode.
- Green appears ONLY as: logo squares, ring 1 + `NOW`, the ring dot, selection color, live-clock accent if used. Never as buttons, focus, or feedback.
- Reduced motion honored everywhere; Lighthouse targets met; view-source contains all copy; no console errors.
